Data Collection

Tether Meeting Scheduler Chrome Extension does not collect or transmit any personally identifiable information to external servers. The extension processes data locally on your device for meeting scheduling functionality.

Local Data Processing

  • Calendar availability is read locally to suggest meeting times
  • Email content is analyzed locally to detect meeting coordination
  • Extension settings are stored locally in your browser
  • No personal data is sent to external servers

Permissions Usage

The extension uses minimal permissions:

  • Storage: To save your preferences locally
  • Active Tab: To detect meeting coordination in emails
  • Host Permissions: To work with Gmail, Outlook, and calendar services

Third-Party Services

When you authorize calendar access, the extension uses official APIs:

  • Google Calendar API (with your permission)
  • Microsoft Outlook API (with your permission)
  • OpenAI API (only if you provide your own API key)

Data Security

All data processing happens locally on your device. We implement security measures including:

  • Local-only data processing
  • Secure API authentication
  • No server-side data storage
  • User-controlled permissions

Your Rights

You can:

  • Revoke calendar permissions at any time
  • Clear all extension data by uninstalling
  • Disable specific features in settings
  • Control which data the extension accesses

Chrome Web Store Compliance

This extension complies with Chrome Web Store User Data Policy. We use minimal permissions and process data locally for meeting scheduling functionality only.

Updates

We may update this policy to reflect changes in our practices. Continued use of the extension constitutes acceptance of any updates.
